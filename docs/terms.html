1. Description of Service

DrainShield is a non-custodial Web3 security dashboard developed by IBITI Labs. The Service provides proactive risk analysis for blockchain wallet approvals across supported networks (Ethereum, BNB Chain, Polygon, Arbitrum, Optimism, and Base). Core features include:

  • Approval Auditing: Identifying token allowances and deep contract probes (Proxy, Owner, and behavioral capabilities).
  • Risk Scoring: A multi-factor assessment engine (0-100) evaluating spenders based on exposure and reputation.
  • Threat Intelligence: Real-time cross-referencing against global threat feeds.
  • Panic Mode: Emergency interface for batch revocation of high-risk permissions.

The Service is strictly non-custodial. All security actions are signed locally within the user's connected wallet.
