# Privacy Policy – DrainShield

Last updated: 2026 Volodymyr Podkorytov

DrainShield is a security utility designed to help users monitor wallet permissions and detect potential risks.

## Information We Collect

DrainShield does not collect personal information such as names, emails, or phone numbers.

The app may process the following data locally on the device:

- Public wallet addresses entered by the user
- Blockchain permission data related to those addresses
- Security alert preferences
- Local notification settings

## How Data Is Used

All wallet analysis and security checks are performed using public blockchain data.

DrainShield may use external APIs to fetch publicly available blockchain information in order to detect risky permissions or suspicious activity.

No private keys are ever requested or stored.

## Local Storage

Some application settings may be stored locally on the device using secure storage or local databases.

This data never leaves the device.

## Notifications

DrainShield may send local notifications related to security alerts or monitoring events.

These notifications are generated by the app and are not used for advertising.

## Third-Party Services

DrainShield may use third-party APIs to retrieve blockchain data.  
These services only receive publicly available wallet addresses required to perform security checks.

## Data Sharing

DrainShield does not sell, trade, or share personal data.

## Security

DrainShield is designed as a security tool and follows best practices to avoid collecting unnecessary user information.
